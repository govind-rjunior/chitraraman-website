# Chitra Raman — Static Website Template (Vercel-ready)

## Edit content
- `content/videos.json` — add/edit YouTube videos and categories
- `assets/img/hero.svg` — replace with a real hero image
- `pages/series/*` — edit series pages

## Run locally
```bash
python3 -m http.server 5173
```
Then open http://localhost:5173

## Deploy on Vercel
1. Push this folder to GitHub.
2. Vercel → New Project → Import repo
3. Framework preset: **Other**
4. Build command: *(empty)*
5. Output directory: `.`
